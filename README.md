> 🚧 **Projeto em desenvolvimento ativo** — versão alpha jogável no navegador.

Tamagotchi virtual completo com personalização de sprites, sons chiptune, minigames (quiz, flappy, tetris, dungeon), sistema de saúde/idade, economia, salas compráveis e salvamento local. Jogue direto no navegador.

## 🚧 Status

**O que já funciona:**
- [x] Pet com fome, sono, diversão
- [x] Editor de sprites pixel art
- [x] 4 minigames
- [x] Salvamento local

**Em andamento:**
- [ ] Balanceamento da economia
- [ ] Sons personalizados
- [ ] Modo multiplayer local
- [ ] Versão PWA para instalar no celular

**Roadmap:** v0.4 → loja | v0.5 → conquistas | v1.0 → release

## Como jogar
1. Abra o `index.html`
2. Crie seu pet
3. Cuide e jogue os minigames
